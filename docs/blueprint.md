# **App Name**: RIBATHLIBARARY

## Core Features:

- Book Catalog Search & View: Users can search and filter the library's catalog by title, author, ISBN, and genre, with detailed views for each book.
- Book Management: Librarians can add new book entries, edit existing book details (e.g., title, author, description, ISBN, number of copies), and mark books as available or unavailable.
- Patron Management: Add new library members, edit their personal information, view their borrowing history, and track their active loans.
- Book Checkout & Return: System to process checking out books to patrons, assigning due dates, and recording returns. Automatic status updates for books and patron accounts.
- Overdue Fine Calculation: Automatically calculate and track fines for overdue books based on configured rules, visible in patron profiles.
- AI Book Description Tool: Utilize a generative AI tool to assist librarians by automatically generating concise, engaging book descriptions or summarizing key plot points based on title and author inputs.

## Style Guidelines:

- Color scheme reflects natural, earthy tones for a calm and professional environment. Primary color (Earthy Olive): #485C2E, a deep, desaturated green. Background color (Subtle Linen): #F3F5EE, a very light, desaturated green with high brightness. Accent color (Rich Sepia): #523015, a dark, rich brown for emphasis and interactive elements.
- Headline font: 'Alegreya' (serif) for an elegant, intellectual feel, befitting a library. Body text font: 'Inter' (sans-serif) for clean readability and a modern, neutral look, suitable for displaying large amounts of information.
- Clean, glyph-based icons (e.g., Font Awesome, Feather Icons) for clarity and functionality across actions like add, edit, search, checkout, and return. Icons should maintain the professional and unobtrusive aesthetic.
- A well-organized, responsive grid-based layout for data-dense pages such as book catalogs and patron lists. Clear visual hierarchy to distinguish primary information from secondary details. Use ample whitespace for a clean and professional appearance.
- Subtle, non-distracting animations for transitions between views, form submissions, and feedback on user actions (e.g., successful checkout, overdue warnings). Animations should feel natural and enhance usability without impeding workflow.
