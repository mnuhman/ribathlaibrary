# RIBATH LIBRARY - Modern Library Management

A comprehensive, AI-powered library management system built with Next.js, Firebase, and Genkit.

## 🚀 Live Application
[View Live App](https://6000-firebase-studio-1772277700553.cluster-yylgzpipxrar4v4a72liastuqy.cloudworkstations.dev/)

## ✨ Features
- **Dashboard Overview:** Real-time statistics for total books, issued items, active members, and overdue records.
- **Book Catalog:** Manage your library registry with automated AI-generated book descriptions.
- **Member Management:** Register and track library patrons with batch-based organization.
- **Fines & Penalties:** Automated fine calculation based on configurable daily rates and loan periods.
- **Global Search:** Quickly find books or members across the entire database.
- **Theme Support:** Modern "Deep Ocean Darkness" theme with a clean light mode alternative.

## 🛠️ Tech Stack
- **Framework:** Next.js (App Router)
- **Database/Auth:** Firebase (Firestore & Authentication)
- **AI Integration:** Genkit with Gemini 2.5 Flash
- **UI Components:** ShadCN UI & Tailwind CSS
- **Icons:** Lucide React

## 📦 Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/mnuhman/ribathlaibrary.git
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Set up Firebase:**
   Create a Firebase project and add your configuration to `src/firebase/config.ts`.

4. **Run the development server:**
   ```bash
   npm run dev
   ```
